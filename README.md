# \[기술문서\] 멀티 세션 기반 감성 분석 모델 성능 분석서 V.1
##	개요
본 과제에서는 멀티 모달 대화 내 감성 분석 모델을 개발하였다. 이 문서는 개발한 모델의 성능에 대해 기술한다. 멀티 모달 대화 내 감성 분석 문제는 대화에서 마지막 발화가 가지는 감성을 긍정, 부정, 중립 중 하나로 구분하는 것으로 시각적 정보와 대화의 맥락을 융합적으로 이해하는 능력이 필요하다. 이 기능을 구현하기 위해 시스템-유저 간 패션 대화 데이터셋을 활용하여 CENet 모델을 학습시켰다. 추가로 시각적 맥락과 언어적 맥락의 유무에 따른 성능 변화도 확인하였다.
##	구성 및 범위
구체적으로 본 문서는 다음과 같은 범위와 내용을 기술한다. 
-	감성 분석 문제의 정의 (2. 문제 정의)
-	본 과제에서의 대화 내 감성 분석 문제의 정의
-	학습 데이터셋 설명
-	감성 분석 모델을 위한 학습 모델 설명
-	학습 설정
-	실험 결과 및 모델 성능 분석
-	향후 보완 계획
